# Настройка Inline режима

## Что нужно сделать:

1. Открой [@BotFather](https://t.me/botfather) в Telegram
2. Отправь команду `/mybots`
3. Выбери своего бота
4. Нажми "Bot Settings"
5. Нажми "Inline Mode"
6. Нажми "Turn on"
7. Введи placeholder текст (например: "Введи название мода...")

Готово! Теперь можно писать в любом чате:
```
@твой_бот_username sodium
```

И бот покажет результаты поиска!
